| Q. Diffusion 공정  A   * 입자들의 농도가 높은 곳에서 낮은 곳으로 이동하는 현상 * 고체에서의 확산은 높은 에너지가 필요 * batch type, 웨이퍼 손상 적음 * 등방성 (불순물 profile 정밀 제어 어려움) |

|  |
| --- |
| Q. Ion Implantation 공정  A   * 불순물을 이온화하여 가속시켜 웨이퍼에 주입 * 정밀한 불순물 영역 형성 가능(이방성) * 불순물의 양 및 주입 깊이를 독립적으로 제어 * 불순물 이온은 웨이퍼내로 주입되면서 Si 원자 등과 충돌해 에너지를 잃게 되어 정지   🡪 충돌로 인한 결함 🡪 Annealing으로 치유   * Channeling 효과: 이온이 결정축 방향으로 주입 시 충돌 확률보다 작아 Gaussian 분포보다 깊게 주입 |

| Q. Ion Implant 공정 설비  A   * Ion Source, Beam Line, End station으로 구분   Ion Source   * Filament에 전류를 흘려주어 열전자를 방출시키고 주입된 Dopant Gas와 충돌해 ion을 만들어냄 * 형성된 Ion은 Analyzer를 통해 원하는 원자량을 가지는 target이온만 추출   Beam Line   * 추출된 이온을 가속시키고 wafer에 집속시켜 beam을 형성   End Station   * wafer에 이온이 주입이 되며 beam current를 detect하여 원하는 dose량을 모니터링 |
| Q. Shadowing Effect  A   * Channeling 현상을 개선하기 위해 wafer 특정 입계각의 Tilt를 주어 공정을 진행   🡪 국부적으로 이온이 주입되지 않는 Shadowing Effect가 발생   * Wafer를 rotation 시켜 이온을 주입하거나 후속 annealing처리를 통해 diffusion을 유발하여 해결 |

| Q. Depth Profile Control  A  Nuclear Stopping Mechanism   * 결정을 이루는 원자와 충돌하면서 이온의 에너지가 손실되어 정지하는 것 * 결정 구조에 damage를 줌 * 이온 에너지가 증가할 수록 Nuclear Stopping power는 감소 * 무거운 이온일 수록 지배적   Electronic Stopping Mechanism   * 쿨롱 상호작용에 의한 에너지 손실 * 가볍고 이온 에너지가 높은 이온일수록 지배적 |

| Q. Annealing 공정  A   * Annealing 공정을 통해 에너지를 가진 이온들의 강제 주입에 의한 격자 손상을 회복시키고, 주입된 이온들의 활성화를 위해서이다.   Furnace   * 장시간 열 노출에 의해 Dopant 원자의 확산거리가 길어지면서 넓은 확산 깊이를 갖기 때문에 소자 Scaling과 집적화에서는 매우 취약함   RTA(Rapid Thermal Annealing)   * 짧은 시간에 열처리를 진행하며 확산 및 Profile 제어에 우수   Laser Annealing   * 국소적으로 고에너지 빔을 가해 신속하게 어닐링 * Short Channel Effect를 억제하기 위해 연구 중 * 이온 주입 후나 열처리 후나 이온 분포의 profile이 유사하게 유지된다는 점이 장점 * 국부적인 영역에 고온을 가하기에 박막의 Stress에 의한 이슈가 있음 🡪 overlay 이슈 야기할 수 있음 |

| Q. 이온 주입 공정 이후 평가 방법  A  4-Point Probe   * 간격이 일정한 probe tip 4개를 wafer에 contact시켜 일정한 전률를 가해 voltage drop을 계산함으로써 Sheet Resistance를 평가할 수 있음   Thermal Wave   * implant된 영역은 damage로 인해 반사율이 달라지는데 반사율을 측정해 uniformity를 측정   SIMS   * depth dopant profile, 깊이 profile, dopant concentration을 측정할 수 있음. * 고에너지의 ion beam을 조사해 sputtering에 의해 표면에서 튀어나온 원자를 분석 |
